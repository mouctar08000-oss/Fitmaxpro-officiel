# Guide : Créer un Compte TikTok Pro/Business pour FitMaxPro

## 📱 Étape 1 : Télécharger TikTok

1. **iPhone** : App Store → Rechercher "TikTok" → Télécharger
2. **Android** : Google Play → Rechercher "TikTok" → Installer

---

## 📝 Étape 2 : Créer un Compte

1. Ouvrir l'application TikTok
2. Cliquer sur **"S'inscrire"**
3. Choisir une méthode :
   - 📧 **Email** (recommandé pour un compte pro)
   - 📱 Numéro de téléphone
   - 🍎 Continuer avec Apple
   - 📘 Continuer avec Facebook
   - 📧 Continuer avec Google

4. Créer un **nom d'utilisateur unique** :
   - Suggestion : `@FitMaxPro` ou `@FitMaxPro_Coach`
   - Évitez les chiffres complexes

---

## 🏢 Étape 3 : Passer en Compte Professionnel (GRATUIT)

1. Aller dans votre **Profil** (icône en bas à droite)
2. Cliquer sur **☰** (menu hamburger en haut à droite)
3. Sélectionner **"Paramètres et confidentialité"**
4. Cliquer sur **"Compte"**
5. Sélectionner **"Passer à un compte professionnel"**
6. Choisir le type :
   - **Créateur** (pour influenceurs fitness)
   - **Entreprise** (pour votre marque FitMaxPro)

7. Sélectionner votre **catégorie** :
   - 🏋️ **Fitness & Sport**
   - 💪 **Santé & Bien-être**
   - 📚 **Éducation**

---

## 🎬 Idées de Contenu pour FitMaxPro

### Format court (15-60 sec)
1. **Exercice du jour** - Un mouvement avec explication
2. **Avant/Après** - Transformations de clients
3. **Astuces nutrition** - Conseils rapides
4. **Behind the scenes** - Coulisses de coaching
5. **Réponses aux questions** - FAQ fitness

## 🔗 Lier TikTok à FitMaxPro

Une fois votre compte créé, ajoutez le lien dans l'admin FitMaxPro :

1. Connectez-vous à `https://fitmax-gains.preview.emergentagent.com/admin`
2. Onglet **"Social Media"**
3. Collez votre URL TikTok : `https://www.tiktok.com/@VotreNom`
4. Cliquez **"Save"**

Le lien apparaîtra automatiquement dans le footer de l'application !
