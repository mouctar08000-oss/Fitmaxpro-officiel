1. **Copier-coller le mot de passe** depuis ce document
2. Vérifier qu'il n'y a pas d'espace avant/après
3. Le M est majuscule : `Mouctar2024!`

### Problème 2 : Page ne charge pas

**Solutions** :
1. Vérifier votre connexion Internet
2. Essayer avec un autre navigateur (Chrome, Firefox, Edge)
3. Désactiver temporairement les extensions (AdBlock, etc.)

### Problème 3 : Reste sur la page de login après avoir cliqué

**Solutions** :
1. Ouvrir la console du navigateur (F12)
2. Regarder s'il y a des erreurs en rouge
3. Vérifier que les cookies sont autorisés
4. Essayer en navigation privée

### Problème 4 : Message "Please login with Google"

**Solution** :
- Ce message apparaît si un compte Google existe avec le même email
- Utilisez le bouton "Continue with Google" à la place
- Ou utilisez un autre email
